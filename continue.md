# <div dir=rtl> continue
<div dir=rtl>تستخدم للخروج من دورة واحدة من الـ loop  عندما تحدث حالة او شرط معين ثم تقوم بإكمال الـloop.<div>

### <div dir=rtl> مثال: <div>

<div dir=ltr>

```
for (int i = 0; i < 10; i++) 
{
  if (i == 4) 
  {
    continue;
  }
  Console.WriteLine(i);
}
  ```
